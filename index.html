Para que vários usuários acessem os mesmos projetos, você precisa de um banco de dados gratuito na nuvem. Siga os 3 passos abaixo — leva menos de 5 minutos:

  1. Acesse supabase.com → crie conta grátis → crie um projeto
  2. Vá em SQL Editor → rode o SQL no botão abaixo para criar as tabelas
  3. Vá em Settings → API → copie a URL e a chave anon public abaixo
